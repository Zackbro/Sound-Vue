# SoundClound

> 用Vue/Vuex写的SoundClound客户端，[在线访问](https://zackbro.github.io/Sound-Vue/#/)需要VPN

## 使用

```
1. npm install 

2. npm run dev

3. visit http://localhost:8082/

```

## 功能

### 已实现的功能
* 播放、暂停
* 上一曲
* 下一曲
* 进度控制
* 自动播放
* 分类
* 状态变化

### 待实现的功能
* 音量控制
* 播放列表
* 详情页
* 移动端
